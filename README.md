# React Assignment

This project contains 6 simple tasks built using React.

## Tasks
1. Display personal information using JSX.
2. Show current date and time.
3. Display a greeting message.
4. Render a list of items using `map()`.
5. Create a user card component with props.
6. Display a product list using parent and child components.

## How to Run
1. Install Node.js (https://nodejs.org).
2. Create a React project
3. Add the given code files (`App.js`, `index.js`, and Task1–Task6 in a `components` folder).
4. Run the project
